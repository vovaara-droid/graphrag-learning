# graphrag-learning

Навчальний проєкт з вивчення AI Data Analyst / Knowledge Graph Engineer.

## Мета

Побудувати GraphRAG-систему — поєднання графових баз знань із retrieval-augmented generation для інтелектуального пошуку та аналізу даних.

## Стек

- **Python** — основна мова розробки
- **Neo4j** — графова база даних для зберігання knowledge graph
- **LlamaIndex** — фреймворк для побудови RAG-пайплайнів
